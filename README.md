# Employee Management System (EMS)

This is a **full-stack Employee Management System** that allows managing employee records efficiently. The project demonstrates the use of **SQL for database management**, **Python FastAPI for backend**, and **React.js for frontend development**.

---

## Features

- Add, update, view, and delete employee records
- Search, filter, and sort employees
- Secure communication between frontend and backend using REST APIs
- Structured database design using PostgreSQL
- Dynamic and user-friendly interface with React.js

---

## Technologies Used

- **Frontend:** React.js  
- **Backend:** Python FastAPI  
- **Database:** PostgreSQL  
- **Version Control:** Git & GitHub  
- **Others:** HTML, CSS, JavaScript

---

## Prerequisites

- Node.js and npm
- Python 3.8+  
- PostgreSQL  
- Git
